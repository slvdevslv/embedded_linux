how does printf work..
* printf formats the output and calls write system call.
* write function writes the buffer content to a special device file - monitor device ie stdout as per write().
* Kernel code of write calls the device driver which writes to the monitor and return values go back all the way to application.

A helloworld app(process) printing on screen:
* process is born in kernel space and then switches to user space.
* app calls printf lib call
* printf calls write system call
* process invoking system call switches to kernel/supervisor mode 
* process then runs kernel code, the device driver code and in the process writes the hello text on to monitor.
* After doing this process switches back to user mode and continues..
